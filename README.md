**Note Taker Project**

 *Overview:*

A simple web app to create, edit, and delete notes that are saved in the browser using Local Storage.

**Features:**
1- Create unlimited notes
2- Edit notes directly 
3- Delete notes instantly
4- Auto save while typing
5- Side panel showing creation dates
6- Data persists after refresh using Local Storage
